# CRUD Operations Documentation

This document summarizes the CRUD operations performed on the Book model in the Django bookshelf app. Each operation is documented in its own markdown file.

## Operations Summary

- [Create Operation](create.md): This section covers how to create a new book instance.
- [Retrieve Operation](retrieve.md): This section explains how to retrieve and display book details.
- [Update Operation](update.md): This section describes how to update an existing book's title.
- [Delete Operation](delete.md): This section outlines how to delete a book instance and confirm its removal.

## Conclusion

This documentation provides a comprehensive overview of the CRUD operations for managing books in the Django admin interface.
